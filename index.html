Veebilehe loomine, privaatsed veebilehitsejad

Käesoleva töö eesmärk on õppida veebiarendust ning uurida, millised veebilehitsejad pakuvad parimat privaatsust.

Valiku põhjendus

Mind huvitavad arvutid ja Internet ning loovtöö annab hea võimaluse süveneda nende teemadega. Otsustasin uurida veebilehitsejaid, sest soovin leida millised neist pakuvad parimat privaatsust ja turvalisust. Samuti on veebiarenduse õppimine mulle huvitav.

Töö käik

Õppisin veebiarenduse põhialuseid Edube platvormil, läbides JavaScript Institute kursuse. Selle käigus lõin oma veebilehe, kasutades HTML-i, CSS-i ja JavaScripti, ning hostisin selle tasuta Verceli platvormil. Samuti uurisin, kuidas veebibrauserid suhtlevad serveritega ning millist teavet nad edastavad.

MÕISTED

Järgnev selgitav osa on koostatud tehisintellekti abiga, sest ise ei oska neid mõisteid nii detailselt seletada, kuid tunnen neid hästi.

Veebilehitseja (brauser)

See on programm, mida kasutad Internetis surfamiseks, näiteks Google Chrome, Firefox või Safari. See kuvab sinu ekraanil veebilehti arusaadaval kujul.

Veebileht

Veebileht on Internetis olev dokument või lehekülg, mida brauser kuvab. See võib olla lihtne tekst ja pildid või keerukas, interaktiivne sait, kus saab näiteks suhelda või osta.

Server

Server on võimas arvuti, mis talletab ja jagab Internetis sisu — näiteks veebilehti, pilte või videoid. Kui soovid midagi Internetist vaadata, saadab sinu arvuti päringu serverile, mis vastab ning saadab vajalikud andmed tagasi.

Hostimine

Hostimine tähendab veebilehe või faili hoidmist serveris, et teised inimesed saaksid seda internetis vaadata. Veebilehe avaldamiseks peab see olema paigutatud serverisse, millel on pidev internetiühendus.

Fingerprinting

See on meetod, mille abil veebilehed või ettevõtted püüavad sind Internetis tuvastada — mitte nime järgi, vaid sinu seadme ja brauseri eriomaduste põhjal. See võib sisaldada infot näiteks brauseri tüübi, ekraanisuuruse ja kasutatavate fondide kohta.

Veebi keeled

HTML - Hyper Text Markup Language

See on veebilehe struktuuri keel — see määrab, kus on pealkirjad, lõigud, pildid ja lingid.

CSS - Cascading Style Sheets

CSS vastutab veebilehe kujunduse eest — värvid, fondid, paigutus. Kui HTML on maja raam, siis CSS on selle värv ja sisustus.

JavaScript (JS)

JS muudab veebilehed elavaks ja interaktiivseks — see võimaldab klikkida nuppe, mängida mänge või näha animatsioone.

TÖÖARENG

1. Etapp – teadmiste omandamine

Omandasin veebiarenduse põhiteadmised HTML-ist, CSS-ist ja JavaScript-ist. Kasutasin selleks Edube platvormi, kus saab tasuta läbida erinevaid IT-kursusi, sealhulgas JavaScript Institute kursusi.

2. Etapp – veebilehe loomine

Lõin oma veebilehe, kasutades õppitud keeli. Koodi kirjutasin programmis WebStorm, mis on spetsiaalselt veebiarenduseks loodud mugav tööriist. Kujunduse ja sisu tegin ise, kasutades ka tasuta vahendeid nagu Google Fonts ja Font Awesome.

3. Etapp – veebilehe avaldamine ja andmete analüüs

Avaldasin veebilehe Verceli platvormil, mis võimaldab tasuta hostimist. Külastades veebilehte erinevate brauserite eriversioonidega (nagu telefon ja arvuti), -režiimide ja ka sätetega. Vercel kogub teabet, mida brauserid saadavad saiti külastades. Peale selle info analüüsimist saab järeldada millised veebilehitsejad on privaatsemad.
